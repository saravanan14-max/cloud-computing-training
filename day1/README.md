AWS IAM Users and Groups

Overview

This project demonstrates the creation and management of AWS IAM users and groups. It focuses on organizing users into groups and assigning permissions securely using IAM policies.

Features

Create IAM users

Create IAM groups

Add users to groups

Attach IAM policies to groups

Manage secure access using least privilege

Technologies Used

Amazon Web Services (AWS)

Identity and Access Management (IAM)

Outcome

Successfully created IAM users and groups, assigned users to groups, and managed permissions efficiently while following AWS security best practices.
